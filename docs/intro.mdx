## Modern Financial Advising For Gen-Z

SR50 is a single source for all of the best personal finance information you can find on the internet. 

We take the time to uncover every single strategy, tactic, and unfair advantage related to personal finance, and have organized it into a free and open source documentation website.

:::tip

If you would like to contribute to this website, either go to the [Contributing](community/contributing.md) page, or go directly to our [GitHub](https://github.com/tpascarella/sr50) and submit a pull request.

:::
